OUTPUT: Redirect using HTTP status 302

Redirects to a new location using 302 redirect.

USAGE
<output> <redirect reference="MODEL | SESSION | REFERED">http://somwhere/path/file?query</redirect> <secure>true</secure> </output> 
redirect = URL to redirect to (can be relative), can have more than 1, first existing reference is used
  reference = (optional) can check model or session or redirect back to referer (sic)
secure = (optional) if true then https will be forced, if false http will be forced, if not present uses whatever chanel it was already on

OUTPUT
Redirects using HTTP status 302
EXAMPLES
  Redirect to absolute URL and force https
  
    http://somwhere/path/file?query
    true
  

  Redirect to relative URL and force http (using reported server/port)
  
    /somepath/file?query
    false
  

  Redirect to a URL in the model and force http
  
    /root/redirectUrl
    false
  

  Redirect to a URL in the session data (if element exists)
    or URL in the model if session data (if element exists)
    or referer from HTTP request
  
    /root/redirectUrl
    /root/redirectUrl
