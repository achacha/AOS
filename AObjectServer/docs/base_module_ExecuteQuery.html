MODULE: ExecuteQuery

Execute simple SQL query against a configured database pool
This module does not handle database specific issues and if more that a simple query is needed then database specific module should be written to handle such queries, possibly using stored procedures

USAGE
<module class="ExecuteQuery">
  <sql><![CDATA[select * from aos.global]]></sql>
  <path>data/global</path>
</module>

sql = SQL string to execute
path = path where the result set object will be added in the model (default is /root/ExecuteQuery/query)
