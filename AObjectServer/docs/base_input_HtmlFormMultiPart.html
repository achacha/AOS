multipart/form-data

This is the MIME for a multi-part POST type HTML submit, all the name/value pairs are parsed from the POST and added to the HTTP request and any uploaded files are added as a reference id fir the request context object.

INPUT
HTML multi-part POST name/value pairs

MODEL
/root/REQUEST/url/query-string/
Adds form data to the request and accessed via context.useRequestParameterPairs() shortcut or directly context.useRequestHeader().useUrl().useParameterPairs() both return reference to the same
AQueryString
object
Uploaded data will instead contain the id of the object which can be retrieved using context.useContextObjects()
