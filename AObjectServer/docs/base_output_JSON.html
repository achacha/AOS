OUTPUT: Generate JSON data from XML data

Use the current model and display it as JSON
Will convert both XML attributes and sub-elements to JSON variable/objects

USAGE
<output type="JSON"> <path>/root/someobject</path> <path>/root/anotherobject</path> <path>/root/moreobjects</path> </output>
path - (optional, 0 or more) specifies XML root to use for generating JSON object
If more than 1 path is used or if path points to multiple nodes, then array notation will be used and JSON array will be returned

OUTPUT
JSON notation of the model and a single object or array of JSON objects. See JSON specification for more details.
