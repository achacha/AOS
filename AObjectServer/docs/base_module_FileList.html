MODULE: FileList

Use another directory as a base

USAGE
<module class="FileList"> <path base="static|data|dynamic|absolute"> <recursive>true|false</recursive> </module>
static - use the static directory (default)
data - use the data directory
dynamic - use the dynamic directory
absolute - use absolute base path
recursive false - only display files in the specified directory (default)
recursive true - display sub-directories and allow navigation inside
