MODULE: InsertIntoModel

Insert XML data into the context model

USAGE
<module class="InsertIntoModel">
  <data>
    <foo>Some data</foo>
  </data>
  <path>/import-node</path>
</module>

data = XML data to parse and insert into the model
path = path where to insert the data (default empty string maps into /root)


SAMPLES
dynamic/samples/InsertIntoModel.json.aos.xml
dynamic/samples/InsertIntoModel.xml.aos.xml
dynamic/samples/InsertIntoModel.template.aos.xml
dynamic/samples/InsertIntoModel.xslt.aos.xml

Each of these controllers will insert data into the model but output it in different form, essentially the InsertIntoModel module deifintion is identical in all 4 and only output module is varied to demonstrate that business logic can easily be separated from the view generation portion
