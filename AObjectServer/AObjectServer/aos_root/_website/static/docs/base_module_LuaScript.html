MODULE: PublishInput

Publishes all input name/value pairs to

USAGE
<module class="LuaScript">
  <filename>lua/myscript.lua</filename>
  <path>script</path>
</module>

Executes aos_root/data/lua/myscript.lua and results in:
<root>
  <script>
    <output source="lua/myscript.lua"><![CDATA[something here output from the script file]]></output>
  </script>
  ...
</root>

<module class="LuaScript">
  <script><![CDADA[ print("Hello World!"); ]]></script>
  <path>script</path>
</module>

Results in:
<root>
  <script>
    <output source="Inline"><![CDATA[Hello World!]]></output>
  </script>
  ...
</root>

filename = Filename of the lua script to use relative to the aos_root/data directory
script = inline script to process
path = path where to insert the data (default empty string maps into /root), will create an element called output for each execution
