OUTPUT: Emit XML model data

Use the current model and display it as XML

USAGE
<output type="XML"> <path>/root/someobject</path> <path>/root/anotherobject</path> <path>/root/moreobjects</path> </output> <output type="XML"> <path>/root/REQUEST</path> <clientside-xsl-stylesheet>reqest.xsl</clientside-xsl-stylesheet> </output>
path - (optional, 0 or more) specifies path to lookup in the model, all elements are added to the common element called 'root'
clientside-xsl-stylesheet - (optional) path to client-side XSL stylesheet to add to the XML document that the browser will use in transform

OUTPUT
XML of the model or parts of it.
