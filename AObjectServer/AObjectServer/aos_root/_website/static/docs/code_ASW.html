ASW and ASWNL macros

The purpose of these 2 macros is to wrap const char * as a const AString object. It allows you to reuse the memory without allocation and coopying a string.

This will create an AString object and allocate memory block:
  AString str("test string");


This will call AString::wrap() and create a reference to a const AString object:
  const AString& str = ASW("test string")

AStringWrapper

ASW(s,n)
s - const char * string
n - length of the string to use (doesn't have to be the entire string, can be partial but not greater than actual length of the memory block)

e.g.:
  ASW("foo", 3); // equivalent to const AString("foo") without memory allocation
  ASW("my long string", 2); // results in equivalent: const AString("my") without memory allocation

AStringWrapperNoLength

ASWNL(s)
s - const char * string
Works just like ASWNL except the length is calculated using strlen(s)

e.g.:
  ASWNL("foo"); // equivalent to const AString("foo") without memory allocation
